## 八、雷达图代码实现

```python
from reportlab.graphics.shapes import Drawing, Polygon, String, Line, Circle
from reportlab.graphics import renderPDF
import math

def draw_radar_chart(dims, scores, width=220, height=220):
    """绘制雷达图
    dims: 维度名称列表（8个）
    scores: 对应分数列表（1-5）
    """
    d = Drawing(width, height)
    cx, cy = width // 2, height // 2
    radius = 80
    n = len(dims)
    
    # 绘制同心圆网格
    for r in [radius * 0.33, radius * 0.66, radius]:
        d.add(Circle(cx, cy, r, strokeColor=colors.HexColor('#e2e8f0'), 
                     fillColor=None, strokeWidth=1))
    
    # 绘制轴线
    for i in range(n):
        angle = 2 * math.pi * i / n - math.pi / 2
        x2 = cx + radius * math.cos(angle)
        y2 = cy + radius * math.sin(angle)
        d.add(Line(cx, cy, x2, y2, strokeColor=colors.HexColor('#e2e8f0'), 
                   strokeWidth=1))
    
    # 绘制数据多边形
    points = []
    for i, score in enumerate(scores):
        angle = 2 * math.pi * i / n - math.pi / 2
        r = radius * (score - 1) / 4  # 归一化 1-5 → 0-1
        x = cx + r * math.cos(angle)
        y = cy + r * math.sin(angle)
        points.extend([x, y])
    
    d.add(Polygon(points, fillColor=colors.HexColor('#1e3a8a33'), 
                  strokeColor=colors.HexColor('#1e3a8a'), strokeWidth=2))
    
    # 绘制维度标签
    for i, dim in enumerate(dims):
        angle = 2 * math.pi * i / n - math.pi / 2
        label_radius = radius + 18
        x = cx + label_radius * math.cos(angle)
        y = cy + label_radius * math.sin(angle)
        d.add(String(dim, x, y, fontName='Helvetica', fontSize=8,
                     textAnchor='middle', fillColor=colors.HexColor('#475569')))
    
    return d
```

## 十一、附录：ReportLab 实现注意事项

1. **字体注册**：必须先调用 `register_fonts()` 确保 `CHINESE_FONT` 不为 None
2. **中文换行**：`Paragraph` 默认不支持自动换行，长段落需手动控制宽度
3. **表格样式**：`TableStyle` 支持 `BACKGROUND` 逐格设置颜色
4. **雷达图**：需在 `doc.build(story)` 之前将 `Drawing` 对象加入 story
5. **分页控制**：使用 `PageBreak` 控制分页，或让 ReportLab 自动处理
6. **调试模式**：本地生成 PDF 时建议用 `buffer.getvalue()` 保存到文件检查
